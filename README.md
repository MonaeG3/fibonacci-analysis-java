# Fibonacci Algorithm Performance Analysis (Java)

## Overview
This project compares recursive and dynamic programming approaches to solving the Fibonacci sequence, highlighting performance and efficiency differences.

## Technologies Used
- Java
- Eclipse IDE

## Key Concepts
- Recursion
- Dynamic Programming
- Time Complexity Analysis

## Results
The recursive approach becomes significantly slower as input size increases, while the dynamic programming approach improves performance by storing previously computed values.

## How to Run
1. Open in Eclipse IDE
2. Run FibonacciAnalysis.java
3. View execution time results in console
